Pipeline

in Four Steps

01

Parse SMILES

→
02

Build Graph

Node feature matrix and adjacency edge index are extracted and moved to the compute device.

→
03

The loaded PyTorch model runs a forward pass with the graph data in no-grad evaluation mode.

→
04

Return Result

argmax gives the class label; softmax gives confidence. Results returned as JSON instantly.
